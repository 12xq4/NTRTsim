\subsection{Detailed Description}
The core simulation and base level models. The core directory contains all of the necessary components for modeling and simulation. This includes\-:
\begin{DoxyItemize}
\item the world \hyperlink{classtg_world}{tg\-World},
\item simulation control in \hyperlink{classtg_simulation}{tg\-Simulation},
\item views of the simulation\-: \hyperlink{classtg_sim_view}{tg\-Sim\-View} and \hyperlink{classtg_sim_view_graphics}{tg\-Sim\-View\-Graphics}
\item rendering functions \hyperlink{classtg_bullet_renderer}{tg\-Bullet\-Renderer}, based on \hyperlink{classtg_model_visitor}{tg\-Model\-Visitor}
\item the base class for models \hyperlink{classtg_model}{tg\-Model},
\item components of models such as \hyperlink{classtg_rod}{tg\-Rod} and \hyperlink{classtg_linear_string}{tg\-Linear\-String}
\item the ability to tag models and components with \hyperlink{classtg_tags}{tg\-Tags} and \hyperlink{classtg_taggable}{tg\-Taggable}
\item basic components of controllers \hyperlink{classtg_subject}{tg\-Subject} and \hyperlink{classtg_observer}{tg\-Observer} 
\end{DoxyItemize}
