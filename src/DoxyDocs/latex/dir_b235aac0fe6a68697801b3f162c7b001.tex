\subsection{Detailed Description}
Tools for machine learning on tensegrity structures. Contains an evolution scheme that can be configured for Monte\-Carlo, gaussian sampling or simulated annealing. Also contains adapters to read parameters in and out of evolution objects, and a class to read a simulation configuration from a .ini text file
